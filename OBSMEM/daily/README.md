# Daily

Короткие заметки по дням, которые хронически собираются вокруг текущей работы.

Формат имени:
- `YYYY-MM-DD.md`

Использовать только если нужен дневной журнал, а не durable project page.

## Workflow
- `Current_Work` держит живую сессию в кратком виде.
- `daily/YYYY-MM-DD.md` фиксирует нарратив дня и сохраняет важные повороты.
- `log.md` остаётся хронологической лентой durable milestones.
